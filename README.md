# 🩸 Lifelink – Blood Donation Platform

Lifelink is a smart blood donation platform that connects donors and recipients within a **5–10 km radius** using real-time maps.

This project was developed as a **BCA Final Year Full Stack Project**.

---

## 🚀 Features

• User Registration & Login  
• Admin Approval System  
• Blood Request Creation  
• Emergency Blood Alerts  
• Donor Availability Toggle (ON/OFF)  
• Map View of Nearby Donors  
• Radius Based Search (5–10 km)  
• Notifications System  

---
